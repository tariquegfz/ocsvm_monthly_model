
## 
- This folder contains the yearly updates of OCSVM monthly model for Ae. albopictus. Starting from the 75th percentile model that is based on Kraemer et al dataset and contains the training data from 1975-2014 (ocsvm_albopictus_median_parameters_quantile_75.model).
- The models obtains after yearly updates have the corresponding year in the filename.
